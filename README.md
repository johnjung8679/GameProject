Weather Yacht
=============

날씨 테마를 입힌 요트 다이스 게임입니다. 도시를 입력하면 현재 날씨를 불러와 배경과 특수 찬스가 바뀌고, 최대 4명의 플레이어가 번갈아 주사위를 굴려 점수를 기록합니다.

주요 특징
- IP 기반 자동 위치 감지 및 입력한 도시의 날씨/기온 표시
- 날씨별 찬스 스킬: 맑음(주사위 6 고정), 흐림(선택 주사위 재굴림), 비(+5점 보너스), 눈(주사위 값 교환), 폭풍(모든 주사위 강제 재굴림)
- 카테고리별 점수 예상 표시 및 스코어보드
- 마우스 휠/스크롤 지원: 카테고리와 점수판 세로 스크롤, Shift+휠로 가로 스크롤
- F11 전체 화면, Esc 해제


게임 플레이 요약
- 시작 화면에서 "게임 시작"을 누르고 플레이어 수(1~4)와 이름을 입력합니다. 도시명을 적으면 해당 지역의 날씨를 불러옵니다.
- 각 차례마다 최대 3번 주사위를 굴릴 수 있고, 클릭하여 원하는 주사위를 고정/해제합니다.
- 굴린 뒤 남은 카테고리 중 하나를 선택해 점수를 기록합니다. 모든 카테고리를 채우면 게임이 끝나고 최고 점수가 승리합니다.
- 날씨별 찬스 버튼은 플레이어당 한 번씩만 사용할 수 있으며, 현재 날씨에 따라 효과가 달라집니다.
